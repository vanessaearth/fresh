
# 利用vue-cli构建vue+webpack整体项目
#### 1.//全局安装vue-cli
**npm install -g vue-cli**
#### 2.//初始化项目文件
**### vue init webpack vuedemo(自定义的项目文件名)**
#### 3.//进入项目文件
**### cd vuedemo**
#### 4.//安装模块依赖
**### npm install**
#### 5.//安装sass-load
**### npm install sass-loader --save-dev**
#### 6.//安装node-sass
**### npm install node-sass --save-dev**
#### 7.//安装axios
**### npm install axios --save-dev**
